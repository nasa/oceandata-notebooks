## Summary

This tutorial demonstrates how to access and visualize SeaHawk-HawkEye satellite mission data using Python and the earthaccess package. SeaHawk-HawkEye is a CubeSat mission that provides ocean color observations for studying marine ecosystems and water quality. It walks through the complete workflow from authentication with `NASA Earthdata` to creating geographic visualizations of remote sensing reflectance data.

The focus of this tutorial is specifically on HawkEye Level 2 Ocean Color products, which are atmospherically corrected data products generated by NASA's Ocean Biology Processing Group (OBPG). Users learn to filter data by temporal, spatial, and cloud cover criteria, work with hierarchical satellite data structures using `xarray` datatrees, and create professional visualizations with geographic projections and map elements.

## Learning Objectives

At the end of this notebook you will know:
-How to search for SeaHawk-HawkEye datasets and understand different product collections (L1, L2, validation)
-How to filter satellite data using temporal, spatial, and cloud cover criteria
-How to work with hierarchical satellite data using `xarray` datatrees
-How to extract and visualize ocean color variables like remote sensing reflectance (Rrs)
-How to create geographic visualizations using `matplotlib` and `Cartopy`

```{code-cell} ipython3
results = earthaccess.search_datasets(
    keyword="SeaHawk"
    )

for item in results:
    summary = item.summary()
    print(summary["short-name"])
```

The three collections with SeaHawk/HawkEye are 
1. `HAWKEYE_L2_OC` : This data collection corresponds to HawkEye Level 2 products generated by NASA OBPG L2gen
2. `HAWKEYE_L1` : This data collection corresponds to HawkEye Level 1 products (not atmospherically corrected)
3. `SEAHAWK_VALIDATION` : In situ data collected to validate HawkEye products

We will be using HawkEye L2 products in this tutorial.

+++

### Filter data
The following three options can be specified to filter the data collection using `earthaccess`.
1. `tspan` : Specify date range
2. `bbox`  : Specify spatial extent in decimal degrees
3. `clouds` : Specify cloud cover extent allowed.

```{code-cell} ipython3
tspan = ("2022-11-28", "2024-03-20")
bbox = (-82.75, 30.25, -80.74, 33.25)
clouds = (0, 75)
```

```{code-cell} ipython3
results_scenes = earthaccess.search_data(
    short_name="HAWKEYE_L2_OC",
    temporal=tspan,
    bounding_box=bbox,
    cloud_cover=clouds,
    )
len(results_scenes)
```

## 5. Visualize Hackeye data
#### Load the `Rrs447` variable for visualization

```{code-cell} ipython3
datatree = xr.open_datatree(paths[0])
Rrs_447 = datatree["geophysical_data"]["Rrs_447"]
Rrs_447
```

#### Obtain coordinates from navigation data group and store them along with `Rrs447`

```{code-cell} ipython3
for item in ("longitude", "latitude"):
    Rrs_447[item] = datatree["navigation_data"][item]
Rrs_447
```

#### We can now visualize `Rrs447`

```{code-cell} ipython3
plot = Rrs_447.plot(x="longitude", y="latitude", cmap="viridis", robust=True)
```

#### Add a projection and add a coastline to the visualization

```{code-cell} ipython3
fig, ax = plt.subplots(subplot_kw={"projection": ccrs.PlateCarree()})
im = Rrs_447.plot(x="longitude", y="latitude", cmap="viridis", robust=True, ax=ax)
ax.gridlines(draw_labels={"left": "y", "bottom": "x"})
ax.coastlines()
plt.show()
```
